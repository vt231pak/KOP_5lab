## Початок роботи

### Необхідні компоненти

- Node.js (версія 16 або вище)
- npm або yarn

### Встановлення

```bash
# Встановіть залежності
npm install

# Встановіть перевіювач ліцензій (опціонально, для перевірки сумісності)
npm install -g license-checker
```

## Доступні команди

### Сервер розробки

```bash
npm run dev
```

Запускає додаток у режимі розробки з використанням Vite. Програма відкриється за адресою `http://localhost:5173` (або наступним доступним портом).

### Production Build

```bash
npm run build
```

Будує додаток для продакшену. Оптимізує та збирає програму в один пакет.

### Перегляд Production Build

```bash
npm run preview
```

Переглядає production build локально.

### Перевірка якості коду

```bash
npm run lint
```

Запускає ESLint для перевірки якості коду та узгодженості стилю.

### Перевірка ліцензій

```bash
npm run check-licenses
```

Перевіряє, що всі залежності мають сумісні ліцензії відповідно до GDPR та стандартів відкритого коду.

### Документація

```bash
npm run docs
```

Генерує документацію за допомогою TypeDoc. Документація буде розташована в папці `docs/`.

### Storybook

```bash
npm run storybook
```

Запускає Storybook для інтерактивного перегляду компонентів на портах 6006.

```bash
npm run build-storybook
```

Будує Storybook для продакшену.

## Конфігурація

### Налаштування гри

Налаштування складності та інші параметри гри керуються через Redux зберігання, розташоване в `src/store/slices/settingsSlice.ts`.

### Рівні складності

- **Легко** - 8 карточок (4x2 сітка)
- **Середньо** - 10 карточок (5x2 сітка)
- **Важко** - 12 карточок (4x3 сітка)

### Налаштування анімації

Швидкість перевороту карточек налаштовується через параметр `animationSpeed`:
- `slow` - 0.9 секунди
- `normal` - 0.6 секунди (за замовчуванням)
- `fast` - 0.3 секунди
