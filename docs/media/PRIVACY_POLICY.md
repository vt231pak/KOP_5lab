# Політика приватності та GDPR / User Instructions

## Політика Приватності

**Дата останнього оновлення: 2026-05-21**

### 1. Вступ

Ця Політика приватності описує, як Emoji Cards Game («Додаток») обробляє ваші дані. Ми зобов'язані дотримуватися GDPR та захищати ваші персональні дані.

### 2. Зібрання даних

#### 2.1 Дані, які ми збираємо:
- **Локальні дані:** Статистика гри (кількість ходів, час, рівень складності) зберігаються виключно на вашому пристрої
- **Дані про файли cookie:** Конфігурація згоди на використання cookies (в межах вашого браузера)
- **Технічні дані:** IP-адреса (для функціонування хостингу)

#### 2.2 Дані, які ми НЕ збираємо:
- Персональну інформацію (ім'я, email, адреса)
- Дані про відвідування третіх осіб
- Аналітичні дані з третіх сервісів
- Географічні дані

### 3. Обробка даних

**Локальна обробка:** Всі дані про гру обробляються виключно у вашому браузері. Ми не відправляємо ваші дані на зовнішні сервери.

**Сховище:** LocalStorage вашого браузера - контролюється тільки вами

### 4. Cookies

Додаток використовує cookies тільки для:
- Зберігання вашої згоди на використання cookies (GDPR compliance)
- Налаштувань гри (складність, мова)

**Як керувати cookies:**
- Ви можете прийняти або відхилити cookies через popup при першому відвідуванні
- Можете скинути cookies у налаштуваннях браузера будь-коли

### 5. Права користувача (GDPR)

За GDPR у вас є право:
- **Права на доступ:** Вимагати доступу до своїх даних (в локальному сховищі)
- **Право на видалення:** Видалити свої дані (очистити LocalStorage)
- **Право на портативність:** Експортувати свої дані
- **Право на заперечення:** Не давати згоду на обробку даних

**Як реалізувати ці права:**
1. Відкрийте DevTools браузера (F12)
2. Перейдіть на вкладку "Application" → "Local Storage"
3. Виберіть домен додатку
4. Переглядайте або видаляйте дані

### 6. Безпека

- Всі дані зберігаються локально на вашому пристрої
- Немає передачі чутливих даних на зовнішні сервери
- Використовується HTTPS для зв'язку з хостингом

### 7. Треті сторони

Додаток використовує наступні бібліотеки з відкритим кодом:
- **React** - UI фреймворк
- **Redux Toolkit** - управління станом
- **React Router** - навігація
- **React Cookie Consent** - управління cookies

Всі ці бібліотеки мають вільні ліцензії та не збирають ваші дані.

### 8. Зміни в Політиці

Ми можемо оновити цю Політику без попередження. Перевіряйте регулярно.

## User Instructions / Інструкція користувача

### Як грати

1. **Вибір складності:**
   - Easy (8 карток) - для новачків
   - Medium (10 карток) - для середнього рівня
   - Hard (12 карток) - для експертів

2. **Гра:**
   - Натисніть на карточку, щоб розкрити emoji
   - Знайдіть пару для кожного emoji
   - Спробуйте завершити гру за мінімум ходів

3. **Статистика:**
   - Порахунок ходів показується вгорі
   - Час гри також відстежується
   - Ваші кращі результати зберігаються локально

### Контроль даних

**Очищення даних:**
- Видаліть все локальне сховище: DevTools → Application → Storage → Clear all
- Видаліть тільки дані гри: DevTools → Application → Local Storage → видаліть потрібні елементи

**Експорт даних:**
- Дані доступні у DevTools → Application → Local Storage
- Скопіюйте потрібні значення

### GDPR Права

Ви можете:
- ✅ Переглядати всі свої дані (локальне сховище)
- ✅ Видалити всі дані будь-коли
- ✅ Відхилити cookies
- ✅ Не використовувати аналітику третіх сторін (якої немає)

### Обмеження

- Даний додаток працює тільки в браузерах, які підтримують LocalStorage
- Дані не синхронізуються між пристроями
- Дані видаляються при очищенні cache браузера

### Часті запитання

**Питання: Мої дані відправляються на сервер?**
Відповідь: Ні, всі дані зберігаються тільки у вашому браузері.

**Питання: Як видалити свої дані?**
Відповідь: Открійте DevTools (F12), перейдіть в Application → Local Storage та видаліть дані.

**Питання: Чи відстежує додаток мої дії?**
Відповідь: Ні, немає ніякого відстеження чи аналітики.

**Питання: Як отримати копію своїх даних?**
Відповідь: Використовуйте DevTools для доступу до Local Storage.
